# Quickstart: Phase I - In-Memory Todo App

## Prerequisites
- Python 3.8+

## Running the Application
The application is a standalone script. Run it using:

```bash
python todo.py
```

## Usage
1. On start, you will see a menu:
   - 1. Add Task
   - 2. View Tasks
   - 3. Update Task
   - 4. Mark Complete/Incomplete
   - 5. Delete Task
   - 6. Exit
2. Select an option by typing the corresponding number.
3. Follow the on-screen prompts to enter task details or IDs.

## Limitations
- **No Persistence**: All data is lost once the program is closed.
- **Single Session**: Only one user interaction at a time.
